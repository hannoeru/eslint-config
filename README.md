# ESlint Config

My eslint config preset

## Packages

- basic: basic rules
- vue: vue with typescript
- react: react with typescript
- next: includes react with next eslint config
- all: includes vue and react
- vue-js: vue with javascript

## Development

```bash
# install pnpm
npm i -g pnpm

# install deps
pnpm i

# run release script
pnpm release

# lint
pnpm lint
pnpm lint:fix
```
